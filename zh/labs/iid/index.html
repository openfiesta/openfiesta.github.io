# 简介

“交叉设计空间”是一个创客空间，在这里你可以快速地实现你的想法，实验室的设备和工具会帮助你完成原型的搭建。

这里培养学生的跨学科合作能力、创新思维和设计能力。艺术家，设计师，工程师和产品设计师可以在这里探索更多的可能性。

# 行为准则

  • 不要占用座位或将你的个人物品长时间放置在实验室；

  • 在你学会设备的操作方法之前，不要单独使用，你可以向实验室管理员寻求帮助；

  • 工具使用后请尽快放回原处。

  • 请自行带走留下的垃圾，不要留下零食或饮料。

# 仪器清单

# Arduino开发板

Arduino是一款便捷灵活、方便上手的开源电子原型平台。Arduino开发板能够读取输入——从传感器、按钮或串行通信等，然后通过计算产生输出——例如运行电机、点亮LED灯等。它是一个微控制器，可以用简单的指令来操纵其做出相应的反应。 https://www.arduino.cc/ (opens new window)

# 树莓派 Raspberry Pi

树莓派是一台微型电脑，它预装了Linux系统，所以尽管体积很小，但它可以处理和台式机一样的许多任务。并且它可以连接传感器和执行器，方便地进行嵌入式开发。

# 3d打印机

注意

如果你还不会使用3D打印机，请在实验室管理员的帮助下使用！

警告

请注意喷嘴高温，且机器运动时禁止触碰，以免受伤！

3D打印机，即快速成形技术的一种机器，它以数字模型文件为基础，可以将切片后的三维模型文件，通过逐层打印的方式来构造物体。

# 示波器

用来测量交流电或脉冲电流波的形状的仪器，它能把肉眼看不见的电信号变换成看得见的图像。

# 机械臂

开源的可编程机械臂，具有与人类手臂相似的功能，可以配合其他组件开发出仿人体工学的项目。

警告

以下工具使用时谨防烫伤，用完请断电！

# 焊台

焊台是一种用于焊接的工具，通过给焊料供热，使其熔化，从而使两个工件焊接起来。

# 热风枪

使用气流将发热元件的热量传导到需要拆装的部件上的工具。

# 热熔胶枪

电热融化热熔胶条进行粘接的工具。
