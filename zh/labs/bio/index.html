# 仪器

# 分析天平

厂家：艾德姆ADAM 型号：NBL~214i

分析天平是实验中进行准确称量时最重要的仪器，可精确称量至0.0001g （即0.1mg）以上。

# 立式高压蒸汽灭菌锅

厂家：上海申安 型号：LDZF-75L-II

实验室高压灭菌器通过使用高温和高压产生蒸汽来对实验室设备进行灭菌。将器具装入高压釜时，重要的是让器具之间不要相互接触，从而使蒸汽可以接触每个表面以达到灭菌的效果。

# 生物显微镜

厂家：上海永科仪器有限公司 型号：YKJ-660研究型金相显微镜

显微镜是一种获得微小物体放大图像的仪器，以获得一些无法用肉眼看到的细节。本实验室使用的生物显微镜装配了4倍，10倍，20倍和100倍放大镜头。

# 离心机（大）

厂家:湘仪 型号：L530

实验室细胞用离心机可以通过高速离心分离液体样品。离心机的旋转引起向心加速度，并根据密度将物质分离。本实验室的细胞用离心机装配了不同的转子和配套器具，可用于15mL和50mL的离心管装配样品离心。

# 凝胶电泳仪

厂家：BIO-RAD 型号：通用Power Pac

凝胶电泳仪用于根据核酸和蛋白质的大小和携带电荷不同来分离核酸和蛋白质。凝胶电泳仪用于法医，分子生物学，遗传学和微生物学实验室，用于运行和比较DNA样品。垂直凝胶电泳仪比水平仪器可分辨较小的DNA片段，并且可以容纳更大的样品量。

# 无尘通风柜

无尘通风柜不需要任何外部通风。它使用过滤器从循环空气中捕获溶剂烟气，然后将其排入房间。无管排烟柜内的传感器监控气体，如果过滤器过载导致烟雾逸出，会发出警报声。

# 高速分散器(均质机)

厂家：宁波新芝生物科技有限公司 型号：XHF-DY

从组织，植物和食物样品中提取的所有样品都需要在进一步处理之前将样品分解。尽管简单的手动均质也很常见（手动研磨），但均质机可以更彻底，更快速地将样品研磨至目标粒径。

# 实验室制冰机

厂家：宁波新芝生物科技股份有限公司 型号：XB-20

多种实验和反应都需要在一个或多个步骤中进行冰浴，如PCR等。本实验室配备的制冰机既可以制冰又可以储冰。

# 台式恒温振荡器（摇床）

厂家：上海精宏实验设备有限公司 型号：THZ-312

摇床可以应用于包括细胞培养。除了稳定的温度条件外，振动器还以通过改变速度和震动轨迹来影响细胞等培养物的生长。

# 红外接种环灭菌器

厂家：海门市齐林贝尔仪器制造有限公司 型号：MH-3000B

红外线消毒器用于灭菌，可以在安全柜中快速创造无菌条件。它采用无石棉芯材，通过大量加热产生815°C（1500°F）的温度从而在5到7秒钟内完全消毒。

# 注射泵(微量进样泵)

厂家：LongerPump 型号：LSP04-1A

研究用注射泵是研究实验室中对于非常微量的流体进行输送的设备。

# 电热恒温培养箱（细菌）

厂家：邦西仪器科技（上海）有限公司 型号：HN408S

细菌用培养箱，可以调节温度，湿度，二氧化碳浓度等其他环境条件。

# 实验室烘箱恒温鼓风干燥箱

厂家：上海精宏实验设备有限公司 型号：DHG-9070A

主要用于烘烤微流体和其他玻璃仪器用的PDMS芯片。本实验室配备的烘箱可以在40至250°C的温度范围内加热。

# 层流罩

厂家： AIRTECH苏州安泰空气技术有限公司

用于创造无菌条件从而进行细胞培养。请勿用于细菌培养。它可以用于其他生物，例如植物，微流体或制备培养基。

注意

Do NOT use this flow hood with bacterias. It's meant to be used for other organisms as plants, microfluidics or prepare culture mediums. For working with sterile conditions with bacterias use the Bunsen lighters.

# 磁力搅拌器

厂家：IKA 型号：RH-basic

可以帮助固体试剂溶解在溶剂溶解。请在使用后对搅拌磁子进行清洁。

# 微滴分光光度计

厂家：上海宝予德科学仪器有限公司（Bio-DL） 型号：Micro drop

分光光度计的原理相同，但用于分析非常小的体积的液滴（一般在2微升）。通常用于分析DNA / RNA /蛋白质浓度。

# 微波炉

厂家：松下（Panasonic）型号：NN-GM-333W

主要用于琼脂制备。请勿用于加热食物。

# 轨道振动器

厂家：海门市其林贝尔仪器制造有限公司 型号：TS200

用于不同的场合的样品制备和孵育（Western转印，ELISA等...）。

# 热循环仪（PCR仪）

厂家：BioRad 型号：T100

这款BioRad PCR仪可以精确扩增遗传物质（DNA / RNA）。

# pH计

厂家：雷磁 型号：PHS-3E

PH计，可以用于测量溶液中氢离子浓度（酸度或碱度）的电子设备。从根本上讲，pH计由连接到pH响应电极和参比电极的电压表组成。

# 等离子清洁机

厂家：铭恒 型号：PDC-MG

通常用于活化材料表面。在我们的实验室中，主要作用是进行PDMS表面活化，使其随后与其他表面共价结合。

# 四维旋转混合仪

厂家：海门市其林贝尔仪器制造有限公司(Kylin-Bell Lab Instrument) 型号：BE1100器

旋转混合器可轻柔至剧烈地混合多达50ml的实验室样品。旋转混合器具有可调节的速度和倾斜角度，可以用于混合血液样本及制备均匀分散溶液。

# 冷冻离心机

厂家:湘仪 型号：TGL-16

该机器可以在低温下通过密度的差异来分离混合物。配套1.5 mL微量离心管。

# 紫外可见分分光光度计

厂家：上海菁华科技有限公司 型号：752N

用于根据其不同化合物的光吸收模式分析稀释液的组成。

# 迷你离心机

厂家：IKA 型号：IKA MINI G S025

迷你离心机，配备两种离心转子和多种试管套，适用于2.0ml，1.5ml，0.5ml，0.2ml离心管和PCR用0.2ml，8排离心管离心。具有开盖暂停功能能，合盖即自动在原时间继续运行。

# 匀胶机（旋涂仪）

厂家：北京赛德凯斯电子有限责任公司 型号：KW-4B

利用离心力，使液体试剂均匀地覆盖表面。

# 超纯水仪

厂家：Rephile 型号：RD0A04000仪

该机器有两种模式。可以产生反渗透级水和超纯水。

# 超声波清洗机

厂家：宁波新芝生物有限公司 型号：SB-100DT

利用超声波对设备进行清洗的仪器。

# 真空泵

真空泵是一种从密封或封闭的容器中除去气体分子以留下部分真空的泵。

# 电热恒温水浴锅

厂家：上海精宏实验设备有限公司 型号：DK-S24 温度范围5~99℃

可以提供恒温水浴环境。本实验室配备的水浴锅具有无缝不锈钢内部和环氧涂层外部，可抵抗腐蚀和化学损害。

# 涡旋震荡仪

厂家：IKA 型号：V2 S025

可以定时，调速，接触式振荡，能够把各种试管，离心管或者微孔板的溶液在可控条件下迅速混合均匀

# 投射反射分析仪

厂家：上海驰唐实业有限公司 型号：WFH-203D

WFH-203B暗箱式三用紫外分析仪采用紫外线灯管及滤光片组成。一机能分别发出强烈的253.7nm和365.0nm紫外线，亦能同时一次性混合使用两种波长，它可适用于DNA、RNA电泳凝胶样品的观察，还可检测蛋白质，核苷酸等。

# 恒温金属浴锅

厂家：IKA 型号： DBH2 S025

采用微电脑控制的恒温装置，控温精度高，制样平行性好，以替代传统的水浴锅等类似装置，广泛用于样品的保存孵育、反应、DNA提取、RNA提取、电泳变性等实验。

# 基因导入仪

厂家：宁波新芝生物有限公司 型号：SCIENTZ-2C

基因导入仪广泛应用于各种动物、植物细胞和微生物的电穿孔，也可作细胞杂交、融合、基因导入的研究。为了提高基因受体细胞导入率及存活率，在真核细胞(如小鼠细胞和人类细胞等哺乳动物)导入基因时，须加入特殊的电缓冲液，电转缓冲液可另行购买，在做大肠杆菌等原核细胞和酵母时可以不用以上特殊缓冲液。

# 医用冷藏冷冻箱(医用冰箱)

厂家：海尔 型号：HYCD205

可用于储存疫苗、药剂、血液等，冷藏温区小、冷冻深度深，冷藏2～8℃，冷冻-20℃～-40℃
